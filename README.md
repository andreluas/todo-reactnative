# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/andreluas/todo-reactnative.git
```

```bash
# Install Dependencies
$ npm i

# Run Aplication
$ npm run start
```

Go to http://localhost:5173/ to see the result.
